Bem-vindo ao Pluk!
Um projeto universitário sendo desenvolvido como atividade da matéria de Programação Mobile II.
Pluk seria um "site aplicação", utilizando a tecnologia PWA, e API's para realizar seus processos.

O Objetivo do Pluk é guiar o usuário para que ele saiba quanto de luz e água sua planta precisa, além de poder pesquisar caracteristicas do tipo da planta e possiveis problemas que ela possa ter.
